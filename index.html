HTML (HyperText Markup Language)
это система верстки, которая определяет, как и какие элементы должны располагаться на веб-странице.
CSS (Cascade Style Sheet)
это то, как HTML представлен. Точно также как HTML описывает контент, таблицы стилей определяют, как документ выглядит.
Javascript
это язык программирования, с помощью которого веб-страницам придается интерактивность.
